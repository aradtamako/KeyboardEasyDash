# KeyboardEasyDash
This tool allows EasyDash to be used with a keyboard only without requiring a game controller.
It works by treating the left and right keys on the keyboard as input from a game controller when pressed.

## Prerequirement
- DirectInput based game controller required
  - Tested devices
    - [BSGP1601BK](https://www.buffalo.jp/product/detail/bsgp1601bk.html)
    - [VJoy Virtual Joystick Driver v1.2 (Driver Only)](https://www.baker76.com/vjoy/)
